## Features

- AI-driven chatbot responses
- Real-time chat interface
- Modern Angular UI (Angular 17+)
- Responsive design for desktop and mobile
- Easily extensible for new AI models
- Example end-to-end and unit testing setup

## Getting Started

These instructions will help you get a copy of the project up and running on your local machine for development and testing purposes.

## Installation

```bash
# Clone the repo
git clone https://github.com/franakol/Angular-Ai-chatbot-app.git
cd Angular-Ai-chatbot-app

# Install dependencies
npm install
```

## Usage

```bash
# Start the development server
ng serve
```

- Go to `http://localhost:4200/` in your browser.
- The app will automatically reload if you change any source files.

## Configuration

- Configure AI service credentials or endpoints in the `environment` files if required.
- Review documentation for any additional environment variables.

## Development

- Generate Angular components:
    ```bash
    ng generate component component-name
    ```
- Build the project:
    ```bash
    ng build
    ```
- Source files are in the `src/` directory.

## Testing

- Unit Tests:
    ```bash
    ng test
    ```
- End-to-End Tests:
    ```bash
    ng e2e
    ```
